Provides classes and interfaces for writing UI tests using the WindowTester API.

WindowTester test cases are subclasses of {@link com.windowtester.runtime.common.UITestCaseCommon} which is a special JUnit test case, tailored for running User Interface (UI) tests. These test cases interact with the UI using an instance of {@link com.windowtester.runtime.IUIContext}.
